# 🚛 Transporte Emporium - Sistema de Control de Flota

Sistema web para gestión de viajes, GPS en vivo, integración con Discord y exportación a Excel.

## Características

- 🔐 Login con PIN para coordinador y choferes
- 🚛 Registro de viajes con punto a punto
- 🗺️ GPS en tiempo real desde el celular del chofer
- 📸 Sistema de fotos con categorías
- 💬 Notificaciones automáticas a Discord
- 📊 Dashboard con costo por kilómetro
- 📥 Exportación a Excel con resumen mensual

## Tecnologías

- React 18 + Vite
- Tailwind CSS
- Leaflet (mapas)
- SheetJS (Excel)
- Lucide Icons

## Desarrollo local (opcional)

```bash
npm install
npm run dev
```

## Despliegue

Esta app se despliega automáticamente en Vercel cuando se hace push a la rama principal.
